# 智学通 · 使用说明书

> 版本: 2.0.0 | 适用: 湖南省大学生程序设计竞赛 应用开发赛道

---

## 一、软件简介

**智学通**是一款面向大学生的AI智能学习助手，集成了大语言模型对话、学习计划自动生成、知识图谱可视化、学习笔记管理等功能，致力于提升学习效率。

### 运行环境
- 操作系统：Windows 10/11、macOS、Linux
- Python版本：3.9 及以上
- 硬盘空间：50MB 以上
- 网络：配置AI API时需要联网，离线模式不需要

---

## 二、安装步骤

### Windows用户

#### 方法一：一键安装（推荐）
1. 双击 `setup.bat`
2. 等待依赖安装完成
3. 双击 `run.bat` 启动程序
4. 浏览器访问 `http://localhost:5000`

#### 方法二：手动安装
1. 确保已安装 Python 3.9+（https://www.python.org/downloads/）
2. 打开命令提示符（cmd），进入程序目录
3. 执行：`pip install -r requirements.txt`
4. 执行：`python app.py`
5. 浏览器访问 `http://localhost:5000`

### macOS / Linux用户
```bash
cd zhixuetong
pip install -r requirements.txt
python app.py
```

## 三、功能使用说明

### 3.1 登录与注册

**演示账号**：用户名 `demo`，密码 `demo123`

首次使用请点击「注册」创建自己的账号：
1. 输入用户名（至少3个字符）
2. 输入密码（至少6个字符）
3. 可选填写邮箱
4. 点击「注册」，然后登录

### 3.2 学习仪表盘

登录后进入仪表盘，可以看到：
- **统计卡片**：笔记数量、学习计划、坚持天数
- **快捷功能**：AI问答、制定计划、知识图谱、记笔记
- **学习趋势图**：本周每日学习时长和完成任务数
- **学习小贴士**：每日更新学习方法建议

### 3.3 AI智能问答

1. 点击左侧菜单「AI智能问答」
2. 在输入框输入问题，**按Enter发送**（Shift+Enter换行）
3. AI会用Markdown格式回复，支持：
   - 代码块高亮
   - 表格
   - 列表
   - 数学公式

**快捷提问**：点击页面上的建议问题或右侧主题按钮

### 3.4 学习计划生成

1. 点击「学习计划」
2. 选择或输入学科名称
3. 描述学习目标
4. 填写可用时间（如"每天2小时，共4周"）
5. 选择当前水平
6. 点击「AI生成学习计划」
7. 等待几秒，AI生成的结构化计划会自动显示

### 3.5 知识图谱

1. 点击「知识图谱」
2. 查看内置的计算机科学知识体系图
3. 操作方式：
   - **拖拽**移动节点
   - **滚轮**缩放
   - **点击节点**查看详情
   - **点击「AI扩展」**获取更多知识点
4. 右侧可点击「知识点标签」快速定位

### 3.6 学习笔记

1. 点击「学习笔记」
2. 点击「新建笔记」
3. 输入标题、选择学科、添加标签
4. 在内容区编写笔记（支持Markdown格式）
5. 点击「AI智能总结」自动提炼要点
6. 点击「保存笔记」

## 四、AI API 配置（可选）

系统内置了离线智能回复系统，**不配置API也能正常使用**。

如需更强的AI能力：

### 获取免费API Key
1. 访问 https://platform.deepseek.com/
2. 注册账号（手机号即可）
3. 进入「API Keys」页面
4. 创建新Key，复制

### 配置方法
1. 打开 `config.py`
2. 将复制的Key填入 `api_key` 字段：
```python
AI_CONFIG = {
    'api_key': 'sk-xxxxxxxxxxxxxxxx',  # 你的Key
    # ...
}
```
3. 重启程序

## 五、常见问题

**Q: 启动后浏览器访问不了？**
A: 检查控制台是否有报错，确认5000端口未被占用。

**Q: 为什么AI回复不够智能？**
A: 系统当前运行在离线模式，回复内容为预设。配置DeepSeek API Key后即可获得真正的AI能力。

**Q: 数据存在哪里？**
A: 所有数据存储在 `data/zhixuetong.db` SQLite数据库文件中。

**Q: 如何重置数据？**
A: 删除 `data/zhixuetong.db` 文件后重启程序即可。

**Q: 能否在手机上使用？**
A: 在同一WiFi下，手机浏览器访问 `http://[电脑IP]:5000` 即可。
